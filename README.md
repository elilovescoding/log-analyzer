# Log Analyzer

This is a Python program that analyzes log files to detect failed login attempts and system errors.

## Features
- Detects failed login attempts using multiple patterns
- Identifies system errors from log entries
- Handles inconsistent log formatting

## Technologies Used
- Python

## How to Run
1. Run the Python script
2. The program will read the log file
3. It will output the number of failed logins and errors

## Example Output
Failed logins: 2
Errors: 3
